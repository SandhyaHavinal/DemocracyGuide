# DemocracyGuide — Interactive Election Assistant

DemocracyGuide is a modern, responsive, and multilingual web application designed to guide citizens through the entire election process. The platform features an interactive timeline, a comprehensive resource directory, and an intelligent chatbot assistant to answer voting-related questions.

## ✨ Features

- **🌍 Multilingual Support:** Instantly toggle the interface language between English (en), Hindi (hi), Tamil (ta), and Telugu (te) without page reloads.
- **🗺️ Interactive Process Timeline:** A visually engaging step-by-step wizard guiding voters from voter registration to election results certification.
- **📚 Resource Dashboard:** A filtered repository of non-partisan voting guides, polling location tools, and official election portals (including specific sections for US and India).
- **🤖 Smart Election Chatbot:** A built-in assistant that utilizes **Fuse.js** for fuzzy matching. It accurately detects user intent even with spelling mistakes to provide relevant, localized answers regarding deadlines, registration, mail-in voting, and polling locations.
- **♿ Accessible Design:** Built with semantic HTML and ARIA live regions to ensure an inclusive experience for screen-reader users.

## 🛠️ Technology Stack

- **Core:** HTML5, CSS3, Vanilla JavaScript
- **Styling:** Custom CSS variables for a dynamic, glassmorphic design theme. Font integration via Google Fonts (Inter, Outfit). Icons by Boxicons.
- **Libraries:** [Fuse.js](https://fusejs.io/) (via CDN) for lightweight, dependency-free fuzzy string matching.
