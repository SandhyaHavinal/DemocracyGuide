# DemocracyGuide — Interactive Election Assistant

DemocracyGuide is a modern, multilingual, and responsive web application designed to help citizens understand and navigate the election process. The platform provides interactive voting guidance, official election resources, and an intelligent chatbot assistant for answering election-related queries.

---

## ✨ Features

- 🌍 Multilingual Support (English, Hindi, Tamil, Telugu)
- 🗺️ Interactive Election Timeline
- 📚 Election Resource Dashboard
- 🤖 Smart Chatbot with Fuse.js Fuzzy Search
- ⏳ Live Election Countdown Timer
- ♿ Accessible & Responsive Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Fuse.js
- Boxicons
- Google Fonts
- Google Gemini API for AI Chatbot 
- Google Maps Embed API for Location Integration
---

## 🚀 Setup

Clone the repository:

git clone https://github.com/your-username/democracyguide.git

To enable AI chatbot functionality:

1. Visit  
   https://aistudio.google.com/

2. Generate a free **Google Gemini API Key**

3. Open:

```plaintext
global.js
```

4. Replace the placeholder key:

```javascript
const GEMINI_API_KEY = "YOUR_GEMINI_API_KEY_HERE";
```
Open index.html in your browser or run with a local server.
